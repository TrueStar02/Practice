# 生成命题公式

假定用`~, /\,\/,->, <->`分别表示否定、合取、析取、蕴含和双蕴含连接词，根据教材定义2.1，生成包含n个命题逻辑词而且只包含三个命题变元p,q和r的命题公式。

例如，包含0个连接词的命题公式：
```
p
q
r
```
包含一个连接词的公式：
```
(~p)
(~q)
(~r)
(p/\p)
(p\/p)
(p->p)
(p<->p)
(p/\q)
(p\/q)
(p->q)
(p<->q)
(p/\r)
(p\/r)
(p->r)
(p<->r)
(q/\p)
(q\/p)
(q->p)
(q<->p)
...
(r/\p)
(r\/p)
(r->p)
(r<->p)
(r/\q)
(r\/q)
(r->q)
(r<->q)
(r/\r)
(r\/r)
(r->r)
(r<->r)
```
***警示：先写一个伪代码算法，然后再实现。你必须用一个独立的函数实现核心算法！***
