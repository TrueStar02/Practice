# 生成不定方程解(1)

### 问题
给定一个形如$x_1 + x_2 + ... + x_n = r$的不定方程，按照字典序列出该方程的所有解。

### 输入
+ 第一行是一个正整数$n$（&le; 5），表示变量个数
+ 第二行是正整数$r$，表示等式右边的值
+ 第三行是$n$个非负整数，表示各个变量的下界约束

### 输出
按照格式要求输出该方程满足约束的所有解

### 输入样例
```
3
6
1 0 0
```
表示方程$x_1 + x_2 + x_3 = 6$, 满足约束$x_1 \ge 1$, $x_2\ge 0$, $x_3\ge 0$。

### 输出样例
```
(1,0,5)
(1,1,4)
(1,2,3)
(1,3,2)
(1,4,1)
(1,5,0)
(2,0,4)
(2,1,3)
(2,2,2)
(2,3,1)
(2,4,0)
(3,0,3)
(3,1,2)
(3,2,1)
(3,3,0)
(4,0,2)
(4,1,1)
(4,2,0)
(5,0,1)
(5,1,0)
(6,0,0)
Total solutions:21
```
