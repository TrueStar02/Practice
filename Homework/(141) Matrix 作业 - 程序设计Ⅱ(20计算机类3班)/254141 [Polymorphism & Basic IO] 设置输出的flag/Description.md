# [Polymorphism & Basic I/O] 设置输出的flag

# 设置输出的flag

# Description
<p>将输入的数按照系统默认格式，定点格式显示浮点数。</p>
<p>&nbsp;</p>

# Input
<p>第一行N代表接下来会有N个cases。</p>
<p>接下来N行中，每行包含一个浮点数。</p>

# Output
<p>对每个浮点数按照 系统默认格式，定点格式分行显示。</p>

# Sample_Input
```
4
2123.324423
2123.324423
2123.324423
123.0000

```

# Sample_Ouput
```
2123.32
2123.324423
2123.32
2123.324423
2123.32
2123.324423
123
123.000000

```

# Hint
<p>重设格式</p>
